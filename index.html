About this repository

What this space is for

This GitHub repository is the working home for Te Mātāpuna's AI learning modules. It does three things: it hosts the resources as live web pages, it keeps a full version history of every change, and it gives the team a shared space to develop and iterate together.

Everything here is a working draft. Resources are built for specific courses and cohorts, tested in the room, then refined. The version history means we never lose earlier iterations — and we can always see what changed, when, and why.

Host
Live web pages
Every HTML file in this repo is instantly available as a live URL via GitHub Pages. No server needed.
Track
Version history
Every upload and edit is recorded. You can see the full history of any file, compare versions, and restore earlier ones.
Collaborate
Team editing
Collaborators can upload updated files directly. Changes are tracked by who made them and when.
How to update or add resources

For the team — making changes

You don't need to know how to code to contribute. If you have an updated HTML file, you can upload it directly and it replaces the old version — while GitHub keeps the history of both.

1
Get added as a collaborator
Ask Craig to add your GitHub username via Settings → Collaborators. You'll get an email invite — accept it and you'll have access to upload and edit files directly.
2
Upload a new or updated file
Go to the repo → Add file → Upload files. Drag your file in. If it has the same filename as an existing file, it will replace it. If it's new, it'll be added alongside.
3
Write a commit message
Before clicking Commit changes, write a brief note about what you changed — e.g. Updated Card 5 scenario wording or Added HOSPO507 game. This becomes the version history entry and helps everyone track what changed.
4
The live URL updates automatically
Within a minute or two of committing, GitHub Pages rebuilds and the live URL serves the new version. No manual republishing needed.
5
To see version history for any file
Click the file in the repo → click History (top right of the file view). You'll see every commit that touched that file, who made it, and when. Click any commit to see exactly what changed.
